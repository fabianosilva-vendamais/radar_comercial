Como colocar o sistema no ar

Um passo a passo simples, sem programar. São 3 serviços gratuitos e 5 fases. Siga na ordem — leva cerca de 20 minutos no total.

Supabase
O banco de dados (guarda as informações)
GitHub
Guarda o código do sistema
Vercel
Publica o site numa URL
1
Criar a tabela no Supabase
Prepara o banco para receber os dados · ~3 min
a
Entre no seu projeto em supabase.com.
b
No menu da esquerda, clique em SQL Editor e depois em New query.
c
Abra o arquivo supabase-schema.sql (vem no pacote) ou copie o SQL da tela Configurações do app, cole na área de texto e clique no botão Run.
✓
Deve aparecer Success. No rows returned. Pronto, a tabela foi criada.
2
Copiar as duas chaves do banco
É o que liga o site ao banco · ~2 min
a
Ainda no Supabase, clique em Project Settings (engrenagem no rodapé do menu) e depois em API.
b
Copie a Project URL (parecida com https://xxxx.supabase.co).
c
Copie a chave marcada como anon / public (começa com eyJ...). Guarde as duas.
Atenção: use só a chave anon / public. Nunca compartilhe nem publique a chave service_role (secret) — essa é privada.
3
Subir o código no GitHub
Sem comandos — pelo navegador · ~5 min
a
Baixe o pacote do projeto (o cartão de download que enviei no chat) e descompacte o .zip no seu computador.
b
Em github.com, clique em New (novo repositório), dê um nome (ex.: radar-vendamais) e clique Create repository.
c
Na página do repositório, clique em uploading an existing file e arraste todos os arquivos descompactados para a área indicada.
✓
Clique Commit changes. Seus arquivos estão no GitHub.
4
Publicar na Vercel
Coloca o site no ar · ~3 min
a
Entre em vercel.com com sua conta do GitHub e clique em Add New → Project.
b
Encontre o repositório radar-vendamais e clique Import.
c
Não precisa mexer em nada (é site estático, sem build). Clique Deploy e aguarde.
✓
A Vercel mostra a URL pública (ex.: radar-vendamais.vercel.app). Esse é o seu sistema no ar.
5
Conectar o banco no app
Liga tudo · ~1 min
a
Abra a URL pública e vá em Configurações → Banco de dados compartilhado.
b
Cole a Project URL e a anon public da Fase 2 e clique Conectar banco.
✓
Aparece Conectado. A outra pessoa só precisa abrir a mesma URL e conectar igual — os dados ficam salvos na nuvem e sincronizados.
